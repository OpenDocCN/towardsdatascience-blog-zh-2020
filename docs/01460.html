商业智能和数据科学的三大相似之处

原文：https://towardsdatascience.com/the-top-3-similarities-between-business-intelligence-and-data-science-293237f2c174?source=collection_archive---------16-----------------------#2020-02-09

分享自一位曾经担任商业智能分析师的数据科学家

根据美国劳工统计局的数据，到 2028 年，对合格的商业智能分析师和经理的需求预计将飙升至 14% ，而对数据科学家 的需求预计将上升 16 % 。

鉴于所有职业的平均增长率约为 5%，这两个不同的领域似乎有更好的前景。然而，商业智能分析师似乎没有数据科学家受欢迎，因为两者的未来前景非常相似。

如果您希望从商业智能分析师转型为数据科学家，但不确定哪种技能可以转移。

如果你对构建机器学习模型感到厌倦，并开始怀疑商业智能分析师是否是你的更好选择？

这篇文章是给你的。

我以前在这两个领域工作过。虽然这两个领域看起来非常不同，但实际上它们是非常相关的。

在本文中，我将与您分享这两个领域之间的三大相似之处。

1.分析和理解数据

现在，我发现很多人自称是数据分析师。我认为这是因为它听起来很酷。但是，你不难搞清楚，大部分只是报数字，不明白为什么会这样。

例如，他们报告说，与前一周相比，销售额增加了 30%。然而，他们并没有试图去理解是什么原因促成了销售的增长。

如果你想在任何一个领域做好，你必须了解数据。不要只是为了报告而报告，而是要努力弄清楚数据想要告诉你的事实。

在数据科学中，你对数据的理解将决定你的模型有多成功。例如，您正在构建一个预测销售的模型。如果你知道什么是影响销售的最重要的因素或因素的组合，那么你可以将这些因素输入到你的模型中。因此，你的模型将能够给出合理的预测。

在商业智能中，这也是非常重要的。每天，你都会收到大量的业务请求。无论你试图回答业务问题还是提供见解，你都需要对数据有一个具体的理解。例如，在不了解客户旅程的情况下，您无法洞察哪类客户转化更多。

2.使用数据解决业务问题

这两个领域使用的软件或技术差别很大，但它们都是用来解决问题的。

例如，商业智能部门需要预测销售额，以便为每个销售团队分配预算。通过对 excel 表格中的历史数据执行简单的回归，并手动调整一些变量以理解未来事件，只需使用 excel 表格中的一些公式即可完成。

根据我的观察，如果这个问题需要由数据科学团队来解决，使用的工具将会非常不同。例如，他们会尝试一个复杂的机器学习模型或深度学习模型，而这些模型仅使用 excel 是无法实现的。

虽然工具不同，但他们都试图尽可能准确地预测销售额。两个团队的结合实际上会产生更有效率的结果。

我为什么这样说呢？

这是因为商业智能团队的任务是回答问题或提供见解。他们将更多地了解数据，因此，如果两个团队合作，数据科学团队可以节省时间来找出用于训练模型的重要变量。

3.讲故事和可视化

想象一下，你能够找出一些有价值的见解，但你最终绘制了一个无法清楚显示你的见解的图表。

或者，也许你绘制了正确的图表，但你无法与你的顾客或客户沟通，让他们清楚地理解你的见解。

在所有清理和分析数据的艰苦工作之后，您还需要确保能够以清晰的方式呈现您的结果。然而，绘制图表的工具可能不同。例如，商业智能团队使用 Tableau、Excel 或内部构建的商业智能仪表板来绘制图表。

另一方面，数据科学团队将利用 Python 或 R 等编程语言通过编码来可视化。

无论两个团队使用什么工具，最终结果都是用 PowerPoint 幻灯片来展示他们的工作。

最终想法

至此，你将对这两位专业人士的相似之处有一个总体的了解。上面提到的一些例子可能会因公司而异。

如果你想从一个领域过渡到另一个领域，你会知道什么样的技能实际上是可以转移的。

因此，你不需要太担心这两份工作之间的不同技能，你可以培养可转移的技能，或者学习你认为自己目前缺乏的技能。
