开始之前，我们有几个先决条件:

  • GitHub账户。
  • 一个终端。
  • 非常有限的Git和一般HTML编辑经验。(我拒绝称之为“HTML编程”)

我们需要一个GitHub帐户的原因是我们将使用GitHub页面，所以如果你还没有GitHub档案，现在是时候创建它了！

现在，请准备好直接从GitHub中大量复制/粘贴一些分步说明。

步骤1，创建一个存储库

转到 GitHub 和创建一个名为username. GitHub . io的新存储库，其中 username 是你在GitHub上的用户名(或组织名)。

注意:如果存储库的第一部分与你的用户名不完全匹配，它就不能工作，所以要确保它是正确的。
