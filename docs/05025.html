正常列表

Type = "1"

只需在ol 标签中添加type = "1"，这将创建一个由 1、2、3等等组成的有序列表。列表项将用数字编号(默认)。

Type = "A"

只需在ol 标签内添加type = "A"，这将创建一个由 A、B、C等等组成的有序列表。列表项目将用大写字母编号。

Type = "a"

只需将type = "a"添加到ol 标签中，这将创建一个由 a、b、c等等组成的有序列表。列表项目将用小写字母编号。

Type = "I"

只需将type = "I"添加到ol 标签中，这将创建一个有序的列表，包含 I、II、III等等。列表项将用大写罗马数字进行编号。

Type = "i"

只需将type = "i"添加到ol 标签中，这将创建一个有序的列表，包含 i、ii、iii等等。列表项将用小写罗马数字进行编号。
