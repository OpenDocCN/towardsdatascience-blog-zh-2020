## 通过这个有用的期刊、视频和讲座的集合，推进你对机器学习的理解。

无论你的背景或专业水平如何，建立一个跨越统计学、计算机科学和机器学习的知识库是你作为一名数据科学家取得成功的关键。

# 网络资源

**🤯** [**arXiv 健全性**](http://www.arxiv-sanity.com/)**——机器学习学科的特点是 arXiv 预印本形式的新信息特别密集。幸运的是，[深度学习英雄 Andrej Karpathy](https://www.youtube.com/watch?v=_au3yw46lcg) 开发了这个界面，使跟上该领域的最新发展变得更加容易。**

****🔖** [**基础论文**](https://docs.google.com/spreadsheets/d/1PTaFyE2AsgTd0p7A5aHvEw0lLzw-9OXJC8Wa1Bg10ug/edit#gid=0)**——[聚合智力](https://ai.science/)团队的**(下面讨论的 [ML 讲解](https://www.youtube.com/channel/UCfk3pS8cCPxOgoleriIufyg)讲座背后的团队)，这里列出了数据科学领域最有影响力的论文。**

****🎨** [**蒸馏**](https://distill.pub/)**——**谷歌蒸馏的团队制作了一本独特的期刊，为开创性的想法提供了卓越的可视化工具。为了更容易进入机器学习领域，请查看谷歌的[人工智能探索工具](https://pair.withgoogle.com/explorables/)。**

**📚[**paper space AI wiki**](https://docs.paperspace.com/machine-learning/)**——**这是我目前为止找到的最好的 AI 词汇表……评论里的 hmu 如果遇到更好的！**

****📐**[](https://www.shortscience.org/)****——**这个在线论坛专注于通过社区传播思想，为讨论机器学习方面有影响力的论文提供了一个平台。****

****🐼 [**Chris Albon 的博客**](https://chrisalbon.com/)——你曾经想知道如何在熊猫中做的一切，加上机器学习闪存卡供购买。****

****🍪 [**Cookiecutter 数据科学**](https://drivendata.github.io/cookiecutter-data-science/) **—** 不用再寻找模板来建立您的下一个数据科学项目。****

# ****YouTube 频道****

******💻** [**速成班人工智能**](https://www.youtube.com/playlist?list=PL8dPuuaLjXtO65LeD2p4_Sb5XQ51par_b)**——**在这 20 集系列中，贾布里勒展示了机器学习的一些基础知识。速成班团队还制作了一个关于计算机科学的很棒的系列节目，应该是我们所有技术领域的人(是的，可能是每个使用计算机的人)必看的。****

******✨** [**ML 解说**](https://www.youtube.com/channel/UCfk3pS8cCPxOgoleriIufyg)**——**加拿大就像美国的聪明大姐，在你登场之前的几年，她就用自己的才华迷倒了你的高中老师。就在你认为加拿大似乎再完美不过的时候，她去给曼梯·里解释了。这个奇妙的频道以大约一小时的讲座为特色，有助于澄清 ML/AI 中的开创性思想。****

******🖌** [**杰伦·阿拉玛**](http://jalammar.github.io/)**——**在我写这篇文章的时候，杰伦已经制作了一个视频，但我相信这个频道会让你继续关注更多精彩的内容。杰伊是[图解变形金刚](http://jalammar.github.io/illustrated-transformer/)、[首次使用 BERT 的视觉指南](http://jalammar.github.io/a-visual-guide-to-using-bert-for-the-first-time/)以及更令人惊讶的平易近人的内容的不可思议的创造者。****

******👁** [**3 个蓝色，1 个棕色**](https://www.youtube.com/c/3blue1brown/featured)**——**这个频道提供了机器学习领域底层统计概念的绝佳介绍。如果格兰特·桑德森不能让你对学习数学产生兴趣，那就不知道什么能了。****

# ****播客****

****🎙 [**英伟达的人工智能播客**](https://blogs.nvidia.com/ai-podcast/)**——**自 2010 年代中期以来，英伟达一直引领着用于深度学习的 GPU 计算的兴起。这期播客的特点是主持人个性鲜明，令人印象深刻地融合了企业领袖、创业人士和技术更新。****

****🔭 [**数据交换**](https://thedataexchange.media/)**——**本·洛里卡(Ben Lorica)每集都会处理机器学习相关的商业案例。播客的网站提供了优秀的文档，加上总是有趣的“相关内容”，条目可以追溯到洛里卡在奥莱利的时候。****

****⚛︎ [**期刊社数据怀疑论者**](https://dataskeptic.com/podcasts/jclub)**——**这个播客就像是坐在你超级聪明的朋友的学习小组里。凯尔、兰、乔治，有时还有一个客人聚在一起讨论数据科学杂志上的最新文章和相关新闻。****

****🗓[**TWIML 人工智能播客**](https://twimlai.com/)——除了主持一个引人入胜的采访系列，twiml 还开始提供播客收听派对，提供独特的协作学习体验。****

****📈 [**线性题外话**](http://lineardigressions.com/)**——**经验丰富的数据科学教育家 Katie 向企业软件工程师兼双关语爱好者 Ben 讲解统计学中的核心思想。这是开始您的数据科学学习之旅的好地方。还可以看看[凯蒂的 Udacity 课程](https://www.udacity.com/course/intro-to-machine-learning--ud120)——这是我在还在学习 Python 基础知识的时候开始学习 DS 的方式。****
