目标:获得每天所有交付的简单可视化，重点关注不同路线的数量

解决方案 : Python 的 Matplotlib 网格函数

  • 列: 1 列= 1 家商店

  • 颜色=白色: 0 发货

视觉洞察力

  • 路线:我们是否每天都有相同的商店分组？
