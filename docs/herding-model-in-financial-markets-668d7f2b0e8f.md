# **第一部分。型号**

该模型基于两个群体的相互作用:

了解资产“真实”价格的原教旨主义者。把他们视为自助餐型投资者，他们能够把握资产的真正价值

投机交易者对“真实”价格一无所知，而是基于“非理性”情绪进行买卖。

总的来说，市场上有 **N** 个交易者， **Ns** —投机(‘吵闹’)交易者的数量， **Nf** —基本面分析者。

**Nf+Ns = N**

现在最有趣的部分来了:互动。互动意味着原教旨主义者可能会转向投机交易者，反之亦然。
