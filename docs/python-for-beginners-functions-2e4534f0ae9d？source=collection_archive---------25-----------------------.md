# #1 什么是函数式编程？

函数式编程是一种编码范式，其中我们定义做什么，而不是执行动作。这个想法最初来自数学——我们定义函数的输入和我们期望的输出。

函数式编程很少依赖于状态管理——这意味着当我们传递相同的参数时，无论输出多少次都是相同的。它可以被解释为数据正在经历一些转换，最终导致所需的输出。
