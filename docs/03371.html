在这篇文章中，我打算温和地向您介绍计算机视觉中的一些概念。不需要以前在该领域的知识，但你至少应该有 python 和机器学习的基本概念。

那我们怎么做？我们将拍摄某人面部的视频，将其分成帧，检测面部的面部标志(鼻子、嘴……)，并在检测到的标志上绘制一些东西。听起来很简单，对吧？好吧，让我们开始吧。
