介绍

随着时间的推移，我看到越来越多的数据科学职位的招聘信息，尤其是在 LinkedIn 和其他类似的招聘网站上。由于当前的事件，在预期的平静之后，公司已经弄清楚了他们的预算和重点。其中一些公司需要尽快或在不久的将来招聘新的数据科学职位。

成为数据科学家有几个原因。我将强调我成为一名数据科学家的五个主要原因，希望它也能与你成为一名数据科学家的一些原因相一致。

各种技能

正如许多职位都有一套通用的预期技能一样，数据科学也不例外，通常被认为拥有这些技能，我将在下面概述这些技能。当然，还有其他的，但我将重点介绍我作为一名数据科学家在各种公司遇到最多的技能。

  • Python (R)

——备受争议的 Python 与 R 之争通常是有争议的，但最终，这取决于公司已经在使用什么作为他们的主要编程语言。有时，数据科学家可以单独工作，形成模型并直接向利益相关者输出结果，在这种情况下通常更多地参考 R。然而，根据我的经验，使用 Python 可以更容易地与数据工程师和软件工程师进行跨职能合作。这种语言经常用于部署目的，因此，从一开始就使用 Python 会更容易。好处是，在学习数据科学的过程中，你将学习 Python 或 R，这将帮助你获得各种技能，如果你选择了不同的职业道路，如软件开发，这些技能可以更好地支持你。

  • SQL

—数据科学家的另一项热门技能是 SQL。有时，在线课程和大学忽略了强调这种语言对于数据科学家的广泛使用的重要性。它几乎用于我参与的每个项目，因为数据集不是简单地给你的。您必须制作自己的数据集，这涉及到使用 SQL 查询数据库表。像 Python ( 和 R )一样，学习 SQL 不仅对数据科学有用，对数据工程和数据分析也有用。

  • 业务

—虽然这个技能不是编程语言，但它仍然很重要。业务，更确切地说是一个概念，是每个数据科学家都要学习的东西。与 SQL 类似，它在教育环境中的教授也没有达到应有的程度。我所说的业务是指，你需要真正习惯于跳入严格意义上不仅仅是数据科学的情况。企业使用数据科学家来提高流程效率，或者发现未来将改变业务的见解。通常，数据科学教育会非常注重获得最高的准确性，比如对不同类型的客户进行细分。达到 98%的准确率是很好的，但是如果你不能想出一个计划来说明你将如何实现这个模型及其结果，那么你的模型就是无用的。

你需要知道利益相关者，CEO，C-Suite/更高层领导，会问你会用你的结果做什么来改变业务。因此，反过来，你会希望通过各种有针对性的电子邮件将这些客户细分群体应用到营销活动中。然后，您可以创建某种测试来查看电子邮件的执行情况，比如 AB 测试。如您所见，拥有极其精确的模型只是数据科学和业务流程的一部分。反复练习这个业务流程是非常有益的。

  • 统计数据

—在学校，人们更加关注统计学，事实证明，统计学可以为数据科学家解决许多问题。了解统计学对于数据科学家来说至关重要，因为它是机器学习模型的基础。实践方差分析，或人口抽样等，在几种形式的业务中是有用的，比如营销活动，或 AB 测试。

独特性

初看起来，数据科学这个不断发展的领域似乎不像过去那样独一无二了。然而，它仍然是独一无二的，甚至在你将要工作的特定公司中更加独一无二。可能还有其他像安全工程师这样可能更独特的角色，但数据科学是独一无二的。

  • 小编制

为了解释人员少的问题，软件工程师通常会有一到四名数据科学家，即使是一个小型科技公司也可能有近 30 名开发人员。当你的角色是这个独特的时候，你可以学到有价值的技能，接触多个部门，并对你的公司产生重大影响。这并不是说前面提到的其他领域缺乏这些优势，但我确实认为您更有可能遇到数据科学领域的各种业务。最终，你会对自己的日常工作感觉良好。这个好处让我想到了我的下一个观点——影响。

影响

在多家公司担任数据科学家后，我清楚地认识到，即使只有一个项目，也能无限期地影响业务，带来巨大的好处。

数据科学家能够产生的影响是显著的。您可以自动化以前的手动流程，为公司节省数千甚至数百万美元。你可以节省公司的时间，更好地分配时间。你将从事的项目在性质和重要性上各不相同。

例如，我参与了一个项目，该项目高精度地自动化了大部分手工流程。感觉到自己对企业的影响力真的很棒。然而，最好的感觉是你能对社会、健康等产生的影响。有无数种方法可以对数据科学产生积极影响，你的日常工作也不例外。

遥远的

在目前的世界状态之前，远程工作已经是技术角色的普遍优势，尤其是在数据科学领域。不幸的是，有几种类型的职业不能受益于这一点，这是我非常钦佩和感谢的。

如果你喜欢在家工作，那么数据科学将是一个绝佳的机会。有很多工具和平台可以帮助创建一个没有实体办公室的成功环境。您可以使用视频会议、消息传递、项目管理以及版本管理工具。工具包括但不限于:

* Zoom* Slack* GitHub* Jira* Confluence

在家工作对我个人来说是一个巨大的好处。我认为这是一个享受生活的机会。生活在一个交通繁忙的城市可能不是最好的感觉，所以能够完全消除这种感觉是一个巨大的积极因素。

支付

是的，数据科学报酬很高。我想确保我把这个列为最后一个好处，因为它不仅广为人知，而且也不是决定职业生涯的最重要因素。虽然更多的钱是好事，但如果你不喜欢你的领域，那么你会很痛苦。然而，如果你喜欢数据科学，并期望在这方面建立自己的品牌和职业生涯，那么你可以期待有高回报。

根据glass door【2】的数据，数据科学家的平均基本工资是 113，309 美元/年。

当然，各州之间甚至各州的城市之间都有差异，所以根据你居住的地方，你可以预期不同的范围。一些公司每年还会发放高额奖金。因为你的角色非常有影响力，你也可以在一些公司获得股份或股票。

此外，根据工作描述或工作职能的不同，你可以预期工资的变化。协商数据科学家薪酬时要考虑的要点包括但不限于:

  • 技能(SQL、Python、R 等)
  • 资历
  • 你向谁汇报
  • 本科或硕士/博士必修
  • 多年经验
  • 机器学习预期/部署
  • 数据工程预期

摘要

如你所见，成为数据科学家有几个原因，尤其是在 2020 年。成为数据科学家的五大理由是:一路上你将学到的各种技能、在你的公司中的独特性、对你的公司的影响、在家远程工作以及报酬。数据科学可能暂时不会消失，很可能会成为一个更受欢迎的职业。重要的是要记住，数据科学的一些分支，如商业智能、软件工程和机器学习，也是很好的职业。希望你会成为一名数据科学家，至少会亲身体验这五个有益的理由。
