什么是 Datapane？

Datapane 是一个 API ，面向用 Python 分析数据并且需要一种方式来分享他们的结果的人。Datapane 使数据分析师或数据科学家可以轻松地从 plot、markdown、dataframe 等组件生成报告，并在 web 上发布以发送给他人。如果你不了解 Datapane，我写了一篇介绍 Datapane 的短文
