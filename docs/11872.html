通过查看上面的输出，我可以说一些事情。

  1. 国家和密度(P/Km)是对象类型。所以我会把它们分别换成 String 和 integer。
  2. 密度(P/Km)在数值上有额外千个逗号。例 1111。所以让我们先去掉它。
  3. 很少有丢失的值，但是现在我将忽略它们。
# remove extra symbol
population['Density (P/Km²)'] = population['Density (P/Km²)'].replace(',','')

现在让我们更改国家类型和密度值。
