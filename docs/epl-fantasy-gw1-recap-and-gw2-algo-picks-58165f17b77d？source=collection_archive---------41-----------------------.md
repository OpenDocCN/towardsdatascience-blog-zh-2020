正如你所看到的，每个位置都有大量选项可供我们选择，因此我们将把上面的许多统计数据插入 Python 中的 ***优化函数，该函数将 ***输出具有最高预期总积分*** 的团队，给定我们的预算约束和进入我们决策过程的其他指标。在团队选择器代码生效之前应用的一些初步过滤器包括:***

1.  **排除受伤或停赛球员**
2.  **从 FDR 高的队伍中排除玩家**
3.  **在 GW1** 中排除没有固定比赛的队伍中的球员
4.  **同一团队不能有超过 3 名玩家**
5.  **总共必须有 15 名球员(GK=2，DF=5，MD=5，ST=3)**
